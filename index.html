    • The 30-Day Coding Challenge is a personal undertaking to keep your coding skills sharp and completing projects.
    The rules are simple, everyday, for 30 days, code anything you want! Just make sure to push the code to a repository, such as github, at the end of each coding session. This helps keep you accountable and let you see how far you have come.
    Below are different topics/projects to get you started if you would like some ideas. Remember to have fun, and congraduate yourself for completing it!

    Web Design

    A great way to brush up on your web design skills is to create a website. Blogs, like this one, are a popular choice since they offer a lot of flexiblity and is well documented if you have any trouble along the way.

    Web API

    If you're more interested in web development, creating an API is a great way to practice. Practicing pulling data from an established API such as OpenWeatherMap is a good place to start. You can even create your own or contribute to an open source community.

    Data Structures

    Data Structures and algorithms are crucial for making your code more efficient, stable, and reuseable. There are various communities devoted to data structures, one that I recommend is leetcode.
